# 🦸 Classificador de Nível de Herói

Projeto desenvolvido como parte do desafio **Classificador de Nível de Herói** da **Digital Innovation One (DIO)**.

## 📖 Sobre o projeto

O objetivo deste projeto é criar um programa em **JavaScript** capaz de classificar o nível de um herói com base na sua quantidade de experiência (XP), utilizando estruturas condicionais.

---

## 🎯 Regras de classificação

| Experiência (XP) | Nível |
|------------------|--------|
| Até 1.000 | Ferro |
| 1.001 a 2.000 | Bronze |
| 2.001 a 5.000 | Prata |
| 5.001 a 7.000 | Ouro |
| 7.001 a 8.000 | Platina |
| 8.001 a 9.000 | Ascendente |
| 9.001 a 10.000 | Imortal |
| Acima de 10.000 | Radiante |

Ao final da execução, o programa exibe a seguinte mensagem:

```text
O Herói de nome {nome} está no nível de {nivel}
```

## ▶️ Como executar o projeto

1. Clone o repositório:


```bash

```

3. Execute o arquivo:
